Desenvolvimento de um portifólio

Curso de HTML e CSS para portfólio

Na primeira aula, trabalhamos com três alunos e introduzimos o HTML. Ensinamos sobre tags como título, parágrafo, imagens, links e listas, além do conceito de semântica. Também explicamos a estrutura básica dos códigos em HTML e introduzimos o CSS in-line, abordando propriedades como width e height. Os alunos entenderam bem os conceitos apresentados. Para a próxima aula, pretendemos aprofundar o HTML, explorando divs, e iniciar o estudo do CSS.

Desenvolvimento de um portifólio

Curso de HTML e CSS para portfólio

Na segunda aula, introduzimos os fundamentos de CSS. Explicamos conceitos como divs, CSS em suas diferentes formas (inline, interno e externo), seletores, classes, IDs, e elementos como img. Em seguida, ensinamos a aplicar esses conceitos e propriedades na construção do portfólio. Os alunos mostraram bastante interesse pelo tema, e a aula correu de forma positiva. Na próxima aula, daremos continuidade ao CSS e faremos a introdução ao JavaScript.

Desenvolvimento de um portifólio

Curso de HTML e CSS para portfólio

Durante a aula, ensinamos os conceitos de box model, display, margin e padding, além de abordar estilos de lista, flex e gap. Também mencionamos JavaScript, embora sem nos aprofundarmos no tema. A aluna demonstrou uma boa compreensão dos conceitos e tópicos abordados. Avançamos significativamente na criação do portfólio dela, e na próxima aula, planejamos finalizar o portfólio e ensinar como publicar o código no GitHub.
