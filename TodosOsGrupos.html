Minicurso de Redes de Computadores

Participantes: Caio Abreu Pereira, Erick Alves Carlos Pontes, Flávia Alessandra Vitório, Micheli Magalhães Corsino, Samuel Damasceno Cardoso

Descrição: O minicurso de Redes de Computadores será ofertado para pessoas com mais de 15 anos que possuam conhecimentos básicos de informática. Não são necessários conhecimentos prévios de redes.

Curso de Gamemaker - Aprender a programar jogos

Participantes: Breno de Lima Candeu, João Pedro Santos, Lucas Magnani Melo, Matheus Selis Soares, Riquelme Furlan

Descrição: O curso terá como público-alvo adolescentes do 9º ano do ensino fundamental até o 3º ano do ensino médio, com idades entre 14 e 18 anos. Não é necessário possuir conhecimentos prévios.

Desenvolvimento de um portifólio WEB

Participantes: Gustavo pacheco, Gustavo Ancete, Felipe Moreira, Marcelo Favaro, Lucas Honorato

Descrição: O minicurso tem como objetivo introduzir conhecimentos básicos sobre a linguagem de marcação HTML e estilização CSS. Voltado para adolescentes de 14 a 15 anos, sem necessidade de conhecimentos prévios, o curso focará na construção de um portfólio web utilizando essas tecnologias.

Aulas sobre HTML e CSS

Participantes: Barbara Amanda Fialho de Souza, Eloisy Monique Moreira, Gustavo Lemos da Silva, Letícia Alves de Oliveira, Maria Beatriz Taveira Alves Silva, Maria Luiza Elidio Castro, Rebecka Leite Teodoro.

Descrição: Em parceria com a Secretaria Estadual de Educação, serão ofertados dois cursos. O primeiro visa capacitar 50 professores da rede estadual de Votuporanga e região. O segundo atenderá diretamente aos alunos do ensino médio.

Curso de Java

Participantes: Ravi Vendramini de Almeida, Josué Teodoro Moreira, Guilherme Grigolin, Gustavo Bizo Jardim

Descrição:O curso busca capacitar desenvolvedores com conhecimentos práticos e teóricos sobre Spring Boot e seu ecossistema, promovendo habilidades para criar, implementar e manter aplicações Java robustas e escaláveis. Consiste na criação de uma api a partir do spring boot que é um framework do Java. Os alunos vão trabalhar com requisições em um banco de dados Postgres no qual vão poder cadastrar, alterar, visualizar e deletar o conteúdo.

Palestras sobre segurança digital

Participantes: Carlos Eduardo Teles dos Santos, Heitor Pinati Martins, Izadora Macedo da Silva, Marcius Gabriel Scriboni da Costa Arantes, Maria Laura Passos Motta, Camila

Descrição: As palestras buscam promover a conscientização e educação sobre segurança digital para diferentes faixas etárias na comunidade. O projeto se divide em três palestras distintas: para crianças, adolescentes e pais/professores. O objetivo de cada palestra é abordar tópicos relevantes à faixa etária específica de forma clara e envolvente.
